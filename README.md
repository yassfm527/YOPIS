# YOPIS - منصة البث المباشر والفيديو

YOPIS هي منصة متكاملة للبث المباشر ومشاركة الفيديوهات مشابهة لمنصة Kick، تم تطويرها باستخدام Flask. توفر المنصة واجهة مستخدم احترافية وتجربة متكاملة للمستخدمين والمبدعين.

## الميزات الرئيسية

- **نظام مصادقة متكامل**: تسجيل الدخول باستخدام حساب جوجل أو التسجيل المحلي
- **صفحات حساب المستخدم**: ملف شخصي، إعدادات، لوحة تحكم
- **ميزات الفيديو**: رفع، عرض، تفاعل، إحصائيات
- **ميزات البث المباشر**: إنشاء بث، لوحة تحكم البث، دردشة مباشرة
- **البحث والاكتشاف**: محرك بحث متقدم، تصنيفات، توصيات
- **واجهة مستخدم احترافية**: تصميم متجاوب، ألوان موحدة، تجربة مستخدم سلسة

## متطلبات التشغيل

- Python 3.8+
- Flask 3.1.0
- SQLAlchemy 2.0.40
- Flask-Login 0.6.3
- Authlib 1.6.0
- Flask-OAuthlib 0.9.6
- PyMySQL 1.1.1
- وغيرها من المكتبات المذكورة في ملف requirements.txt

## طريقة التشغيل

1. قم بتنشيط البيئة الافتراضية:
   ```
   source venv/bin/activate
   ```

2. تأكد من تثبيت جميع المتطلبات:
   ```
   pip install -r requirements.txt
   ```

3. قم بتعيين متغيرات البيئة اللازمة:
   ```
   export FLASK_APP=src/main.py
   export FLASK_ENV=development
   export GOOGLE_CLIENT_ID=your_client_id
   export GOOGLE_CLIENT_SECRET=your_client_secret
   export SECRET_KEY=your_secret_key
   ```

4. قم بتشغيل التطبيق:
   ```
   flask run --host=0.0.0.0 --port=5000
   ```

5. افتح المتصفح على العنوان: `http://localhost:5000`

## قاعدة البيانات

المنصة تستخدم MySQL كقاعدة بيانات افتراضية. يمكن تكوين اتصال قاعدة البيانات في ملف `src/main.py`.

## المصادقة بجوجل
